Программа для POSIX совместимых ОС.
Компилириуется с помощью утилиты gcc и запускается из командной строки.
Необходимо наличие файла data.csv на компьютере.
Пример:
user@lubuntu:~$ gcc lr1.c -o lr1 -lm
user@lubuntu:~$ ./lr1
